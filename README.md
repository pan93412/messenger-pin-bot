## 設定及啟動 // Setup & Start
> 請注意：這台 Messenger Bot 預期會在自己的個人 Facebook 帳戶運作。\
> Note that this Messenger bot will work on your personal Facebook account in expectation.

```sh
$ npm i
$ cp .env.example .env # 記得修改 .env 的設定喔。
$ npm run start
```
